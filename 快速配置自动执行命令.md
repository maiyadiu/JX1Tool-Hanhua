# Cursor 自动执行命令配置指南

## 方法一：通过 Cursor 设置界面（推荐）

1. **打开 Cursor 设置**：
   - 点击左下角的齿轮图标 ⚙️
   - 或者使用快捷键 `Cmd + ,` (Mac) / `Ctrl + ,` (Windows)

2. **找到 Auto-Run Mode 设置**：
   - 在设置搜索框中输入 "Auto-Run" 或 "agent"
   - 找到 "Agents" 或 "Agent" 相关设置
   - 查找 "Auto-Run Mode" 选项

3. **选择自动执行模式**：
   - 选择 **"Run Everything (Unsandboxed)"** 选项
   - 这将允许 AI 助手自动执行命令，无需手动确认

## 方法二：通过设置文件

如果方法一不起作用，可以尝试手动编辑 Cursor 的用户设置文件：

1. 打开命令面板：`Cmd + Shift + P` (Mac) 或 `Ctrl + Shift + P` (Windows)
2. 输入 "Preferences: Open User Settings (JSON)"
3. 添加以下配置：

```json
{
  "cursor.agent.autoRunMode": "run-everything-unsandboxed"
}
```

## ⚠️ 安全提示

启用自动执行模式后，AI 助手将能够：
- 自动执行所有终端命令，无需确认
- 修改、删除文件
- 安装软件包
- 运行脚本

**请确保：**
- 你信任正在使用的 AI 助手
- 仔细审查 AI 建议的命令
- 在重要项目中使用时要格外小心

## 其他选项

如果不想完全自动执行，还可以选择：
- **"Ask Before Running"** - 每次执行前询问（默认）
- **"Sandboxed"** - 在沙箱环境中自动执行（更安全）

## 注意事项

- 自动执行模式可能会执行潜在的危险命令
- 建议在熟悉的环境中或测试项目中使用
- 如果遇到问题，可以随时切换回 "Ask Before Running" 模式
