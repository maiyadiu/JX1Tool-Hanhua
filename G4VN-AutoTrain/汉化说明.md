## 注意事项

### 1. 窗口标题匹配字符串

以下字符串用于匹配目标程序的窗口标题，**未进行翻译**，因为需要与目标程序的原始窗口标题匹配：

- `"§¨ng ký"` - 用于查找注册窗口
- `"Dang ky"` - 用于查找注册窗口
- `"AutoVLBS 1.9"` - 用于查找主窗口
- `"Login"` - 用于查找登录窗口
- `"Bo qua"` - 用于查找跳过按钮
- `"New version!"` - 用于查找版本更新窗口

这些字符串如果被翻译，将导致程序无法找到目标窗口。

### 2. 编译和运行

1. 使用 Visual Studio 打开 `G4VN-AutoTrain.sln`
2. 选择 Release 或 Debug 配置
3. 编译项目
4. 运行生成的 `G4VN-AutoTrain.exe`

### 3. 编码设置

确保所有 `.cs` 文件使用 UTF-8 编码保存，以正确显示中文字符。

### 4. 测试建议

编译后请测试以下功能：
- ✅ 主界面显示中文
- ✅ 错误消息显示中文
- ✅ 按钮文本显示中文
- ✅ 所有功能正常工作

## 后续工作

如果发现以下情况，可能需要进一步调整：

1. **目标程序更新** - 如果目标程序的窗口标题或按钮文本发生变化，需要更新匹配字符串
2. **字体显示问题** - 如果中文显示为乱码，可能需要调整字体设置
3. **字符串长度** - 如果某些中文字符串过长导致界面布局问题，需要缩短文本
