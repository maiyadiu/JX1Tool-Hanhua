# Git 认证快速配置指南

## 方式 1：使用 Personal Access Token（推荐）

### 步骤：
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 生成并复制 token

### 配置命令：
```bash
# 将 YOUR_TOKEN 替换为你的 token
git remote set-url origin https://maiyadiu:YOUR_TOKEN@github.com/maiyadiu/JX1Tool-Hanhua.git

# 然后推送
git push origin main
```
